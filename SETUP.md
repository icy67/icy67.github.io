# Setup Guide

How to run this project locally.

## Requirements

- [Node.js](https://nodejs.org) (LTS version)
- [VS Code](https://code.visualstudio.com)
- [Git](https://git-scm.com)

## Quick Start

```bash
# 1. Clone the repository
git clone https://github.com/icy67/icy67.github.io.git
cd icy67.github.io

# 2. Open in VS Code
code .

# 3. Open index.html in your browser
# (no build step required — pure HTML/CSS/JS)
```

## With Claude Code (optional)

```bash
# Install Claude Code
npm install -g @anthropic-ai/claude-code

# Start Claude Code in the project folder
claude
```

## GitHub Pages

This site is hosted on GitHub Pages at [icy67.github.io](https://icy67.github.io) and [icy67.com](https://icy67.com).

No build step required. Changes pushed to `main` go live automatically.
