## What is this?

```
> icy67 describe --verbose

[icy67] STATUS:  Non-commercial learning platform
[icy67] TYPE:    Free, AI-powered education project
[icy67] MISSION: Make AI capabilities accessible,
                 discoverable — and actually fun.
[icy67] DETAILS: [REDACTED — nice try]
```

A free, AI-powered learning platform for students in Zurich. Built by a student, supported by his family. Our first project: preparation for the Central Entrance Exam (ZAP) for Gymnasium — wrapped as an adventure game, explained with humor, powered by AI.
