// was wir bauen

Eine Lernplattform, die anders ist

icy67 ist eine kostenlose, AI-gestützte Lernplattform für Schülerinnen und Schüler in Zürich. Unser erstes Projekt: Vorbereitung auf die Zentrale Aufnahmeprüfung (ZAP) fürs Gymnasium — verpackt als Abenteuer-Game, erklärt mit Humor, angetrieben von AI.

Wir glauben: Wenn ein Kind für Kinder baut, entsteht etwas, das Erwachsene alleine nie hinbekommen würden. Etwas, das wirklich Spass macht.

🎮

Zelda-Style Quests

Mathe und Deutsch üben — als Abenteuer. Laufe durch eine Welt, triff Charaktere, löse Rätsel.

🧚

Fairytale AI

Ein AI-Chatbot der Fragen über AI lustig und ehrlich beantwortet. Für Kinder, ohne Angst.

🎓

Gymi-Trainer

AI-generierte Übungsaufgaben auf ZAP-Niveau. Mathe, Deutsch, Aufsatz — lustig erklärt.

🔓

100% Kostenlos

Kostenlos für alle Schüler. Keine Gebühren. Bildung gehört allen — nicht nur denen, die es sich leisten können.

// wer wir sind

Ein Familienprojekt

Hinter icy67 steht ein Schüler aus Zürich und seine Familie. Der Schüler hat die Ideen, die Vision und den Enthusiasmus. Die Familie hilft bei Organisation und Infrastruktur. Zusammen bauen wir mit Claude Code und VS Code an etwas, das Kindern helfen soll.

Der Schüler bleibt anonym — aus Überzeugung. Kinderschutz steht bei uns an erster Stelle. Man kann Grosses bewirken, ohne seinen Namen auf ein Poster zu drucken. Die Idee zählt, nicht der Fame.

quick start
# 1. Node.js installieren (von nodejs.org)
# 2. Claude Code installieren
$ npm install -g @anthropic-ai/claude-code

# 3. Repo klonen
$ git clone https://github.com/icy67/mcp-fairytale-plugin.git
$ cd mcp-fairytale-plugin

# 4. Claude Code starten
$ claude
